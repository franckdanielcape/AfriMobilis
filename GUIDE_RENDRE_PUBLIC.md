# Rendre le Dépôt Public pour GitHub Pages

## Pourquoi rendre public ?

GitHub Pages gratuit nécessite un dépôt **public**.
C'est standard pour les projets open source et MVP.

---

## Étapes

### 1. Allez sur GitHub
https://github.com/franckdanielcape/AfriMobilis

### 2. Settings → General
1. Cliquez sur **"Settings"** (onglet en haut)
2. Faites défiler jusqu'à **"Danger Zone"** (en bas)
3. Cliquez sur **"Change visibility"**
4. Sélectionnez **"Public"**
5. Confirmez avec votre mot de passe

### 3. Activer GitHub Pages
Maintenant vous pouvez :
1. Dans le menu de gauche, cliquez **"Pages"**
2. Sélectionnez **"GitHub Actions"**
3. Cliquez **"Save"**

---

## Sécurité

Même en public, vos données sensibles sont sécurisées :
- ✅ Variables d'environnement dans GitHub Secrets
- ✅ Clés Supabase pas dans le code
- ✅ Authentification requise pour l'app
